## 71. 简化路径
#### 示例1 
输入：path = "/home/"
输出："/home"
#### 示例2
输入：path = "/../"
输出："/"
#### 实例3 
输入：path = "/a/./b/../../c/"
输出："/c"

### 题解说明
1. 先对路径进行分割 用split('/')；
2. 分割完了之后 会有 '' '.' '..' '字符串或者字符'四种情况
3. 当遇到'' '.'的时候什么都不做，遇到'..'就出栈一次,遇到字符或者字符串就进栈。
### 代码javascript版本
```javascript
var simplifyPath = function(path) {
    // 将路径以"/"切分,切分后存在的几种情况 "." "字母" ".." ""多个/就会出现最后一种情况
    let resPath = path.split('/');
    let stack = []
    for (let curPath of resPath) {
        if (curPath === '' || curPath === '.') {
            // 当出现''或者.的时候什么都不做
            continue;
        } else if (curPath === '..') {
            // 当出现..向上翻一层，即出栈
            stack.length && stack.pop();
        } else {
            // 用/切分后总共就三种情况所以剩下的就是字母了直接入栈
            stack.push(curPath);
        }
    }
    return '/' + stack.join('/');
};
```
